# Copasetic POC

This repo shows how [Copasetic](https://github.com/project-copacetic/copacetic) can be implemented in CI pipelines and scripts.

## Quickstart

Run `copa-example.sh` to test the scan & patch flow
```bash
./copa-example.sh
```

```bash
./copa-example.sh alpine:latest
```
